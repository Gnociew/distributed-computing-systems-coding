## 题目描述:

* 蒙特卡罗（Monte Carlo）算法计算圆周率的主要思想如下：
给定边长为R的正方形，画其内切圆，然后在正方形内随机打点，设点落在圆内的概为P，则根据概率学原理：
P = 落在圆内点的数量/正方形内点的数量 = 圆面积 / 正方形面积 ＝ PI * R * R / 2R * 2R = PI / 4。即 PI=4P。
这样，当随机打点足够多时，统计出来的概率就非常接近于PI的四分之一了。请根据蒙特卡洛思想来估计 Pi 的值。
